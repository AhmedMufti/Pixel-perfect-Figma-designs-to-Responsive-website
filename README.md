# Company Website Project

This repository contains the source code for the company website, developed to convert provided Figma designs into a fully responsive, ready-to-use website.
Pages implemented: Landing page + Home page + Latest News + Taleemaat + Submit Parhaiyan.

## Project Overview

**Objective:** Turn specific Figma designs into a pixel-perfect, responsive frontend implementation.

**Figma Design Source:** [View Figma Design](https://www.figma.com/design/lIhFleYy9zRk8c1r0lrrT6/Assignment-1?node-id=1-2&p=f&t=wiOx6tEhWaJrWh5x-0)

## Requirements & Features

### Core Requirements
*   **Pixel-Perfect Implementation:** Faithful reproduction of the provided design assets.
*   **Fully Responsive:** optimized for Desktop, Tablet, and Mobile layouts.
*   **Functional Contact Forms:**
    *   Includes proper error handling (validation).
    *   Includes success confirmation messages upon submission.
*   **Code Quality:** Clean, maintainable code structure.

### Technical Stack
*   **HTML5**
*   **CSS** (Custom & Bootstrap 5)
*   **JavaScript** (Form validation and logic)

## Deliverables

*   Completed website files.
*   Source code and documentation.
*   Instructions for future edits (see below).

## Project Structure

The project has been structured as follows:

- `index1.html`: **Entry Point** - The main landing page with the XYZ logo.
- `inner.html`: **Home Page** - Contains the main hero section, navigation, and content cards.
- `submit.html`: **Submission Form** - Functional form for "Submit Parhaiyan" with JS validation.
- `taleemat.html`: **Content Page** - Educational/Informational content.
- `latestnews.html`: **News Page** - Latest messages and updates section.
- `style1.css` & `footer.css`: Custom styling sheets.
- `script.js`: JavaScript logic for form handling and validation.

## How to Run

1.  Clone or download the repository.
2.  Open **`index1.html`** in any modern web browser to enter the site.
3.  Navigate through the links to view the inner pages and forms.

## Form Implementation Details

As per the requirements, JavaScript has been implemented to handle form logic:
- **Validation**: Fields are checked for input before submission to ensure data integrity.
- **Feedback**: Users receive immediate visual feedback (error messages or success confirmations) upon interacting with the forms.
